# Project 4 FSND

### Setup Project

```bash
  $ git clone https://github.com/mr-karan/Udacity-FullStack-ND004
  $ cd Udacity-FullStack-ND004/Project4/
  $ virtualenv venv
  $ source venv/bin/activate
  $ pip install -r requirements.txt
```

### Running Tests
```bash
    $ cd tournament/
```
